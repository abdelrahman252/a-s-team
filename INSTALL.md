# A's Team Dashboard — Install Guide

## Download

Go to the [**Releases page**](../../releases/latest) and download:

| Platform | File |
|----------|------|
| 🪟 Windows | `A's Team Dashboard Setup.exe` (installer) or `A's Team Dashboard Portable.exe` |
| 🍎 macOS Intel | `A's Team Dashboard-x64.dmg` |
| 🍎 macOS Apple Silicon (M1/M2/M3/M4) | `A's Team Dashboard-arm64.dmg` |

---

## Windows Install

1. Download the `.exe` file
2. Double-click it and follow the installer steps
3. The app appears in your Start Menu and Desktop

---

## Mac Install

1. Download the `.dmg` file for your Mac (x64 = Intel, arm64 = M1/M2/M3/M4)
2. Open the `.dmg` and drag **A's Team Dashboard** into your **Applications** folder
3. Close the DMG window

### First launch on Mac

Because the app is not signed with an Apple certificate, Mac will block it the first time. Fix it in one step:

1. Go to **Applications** folder
2. **Right-click** the app → click **Open**
3. A popup appears — click **Open**
4. Done. From now on it opens normally with a double-click

> You only need to do this once.

---

## Requirements

- **Google Chrome** must be installed — the app uses your real Chrome to run the bot
- No other installs needed — everything else is bundled

---

## First Run

The first time you run the bot, a Chrome window will open and ask you to log in to TikTok Ads. Do this once — your session is saved permanently and the bot will skip login on every future run.

For Khod-Whaat, credentials are entered in the Setup page inside the app.

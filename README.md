# A's Team Dashboard

Automated TikTok Ads + Khod-Whaat analytics dashboard for A's Team.

## Download

Go to the [**Releases page**](../../releases/latest) and download the installer for your platform:

| Platform | File |
|----------|------|
| 🪟 Windows | `A's Team Dashboard Setup.exe` (installer) or `A's Team Dashboard Portable.exe` |
| 🍎 macOS Intel | `A's Team Dashboard-x64.dmg` |
| 🍎 macOS Apple Silicon (M1/M2/M3/M4) | `A's Team Dashboard-arm64.dmg` |

For full install instructions see **[INSTALL.md](INSTALL.md)**.

---

## Requirements

- **Google Chrome** must be installed on the machine running the app
- No other installs needed — everything else is bundled
